Run the craft critique protocol from `references/critique.md`:

1. **See the Composition** — rhythm, proportions, focal point
2. **See the Craft** — spacing grid, typography hierarchy, surface whispers, interactive states
3. **See the Content** — coherent story, real data
4. **See the Structure** — CSS hacks, negative margins, calc workarounds

For each issue found, fix it. Then look again. The first build was the draft — the critique is the design.
